# Automotive Test Case Design Portfolio

This repository shows how I design automotive test cases
based on observed OE behavior.

There is no UX or UI guideline in this project.
Only completed system behavior and OE reference behavior were used.

This portfolio focuses on test logic and coverage,
not screen design or visual validation.


# 자동차 테스트케이스 설계 포트폴리오

이 저장소는 OE 동작을 기준으로 테스트케이스를 어떻게 설계했는지를 보여줍니다.

본 프로젝트에는 UX/UI 가이드가 없으며,
완성된 시스템 동작과 OE 기준 동작만을 기반으로 테스트케이스를 구성했습니다.

화면 디자인이 아닌,
테스트 로직과 커버리지에 초점을 둔 포트폴리오입니다.
